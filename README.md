# OOU Cooperative Accounting System

A complete web-based cooperative management system for the OOU Acctg 2005 Alumni Cooperative Multipurpose Society.

## Features
- Member management
- Savings tracking with late fee calculation
- Loan application and approval workflow
- Investment tracking
- Financial reports
- Member portal with self-service
- Role-based access (Admin, Treasurer, Secretary, Member)

## Technology Stack
- Python Flask
- SQLite
- Bootstrap 5
- JavaScript

## Installation
1. Clone the repository
2. Install requirements: `pip install -r requirements.txt`
3. Initialize database: `python init_settings.py`
4. Run: `python app.py`
5. Access at `http://localhost:5000`

Default admin credentials: admin / admin123
